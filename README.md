# ArchEVA-01
vscode theme with colors inspired by EVA-01 from the anime Neon Genesis Evangelion.

## Installation
<pre>
  $ git clone https://github.com/edudscrc/archeva-01-vscode-theme.git
  $ cd archeva-01-vscode-theme
  $ chmod +x ./install.sh
  $ ./install.sh
</pre>
